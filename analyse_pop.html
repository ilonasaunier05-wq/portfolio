Population active, emploi et chômage en France

Analyse de données INSEE autour de la population active, de l’emploi et du chômage entre 2014 et 2020. Les graphiques sont générés directement dans cette page à partir du fichier JSON exporté depuis R.

Objectif : transformer plusieurs fichiers statistiques en une lecture visuelle des dynamiques d’emploi sur le territoire.

01 / Contexte

Explorer les transformations de l’emploi

Le projet combine données démographiques, lecture géographique et analyse de l’évolution des emplois. Le JSON généré depuis R sert de passerelle entre le traitement statistique et la visualisation web.

Données

Sources INSEE

Fichiers sur population active, statut, catégories professionnelles et secteurs d’activité.

Méthode

Export JSON

Les agrégations sont réalisées dans R, puis envoyées au site sous forme de données lisibles par JavaScript.

Résultat

Graphiques interactifs

La page permet de changer d’année, de thème et de département sans images fixes.

05 / Limites

Ce que l’analyse doit nuancer

Les visualisations simplifient la lecture, mais elles dépendent des agrégations choisies dans le script R.

Codes métiers

Certains graphiques utilisent encore des codes NA38, NA5 ou CS3_29 : ils peuvent être remplacés par des libellés.

Carte

La carte dépend du GeoJSON et des départements présents dans le JSON final.

Interprétation

Un indicateur visuel doit toujours être relié au contexte économique et social.

06 / Bilan du projet

Ce projet montre une démarche complète : préparer les données dans R, produire des agrégations fiables, exporter un JSON, puis construire une interface qui rend les résultats plus intuitifs.

Le passage d’images fixes à des graphiques générés dans le site rend la page plus professionnelle, plus dynamique et plus facile à adapter.

Compétences mobilisées
  • Préparation et agrégation de données avec R
  • Export de données en JSON
  • Datavisualisation web avec JavaScript
  • Carte SVG à partir d’un GeoJSON
  • Interprétation d’indicateurs démographiques
